8、列表写法

无序列表(-+*后面空格加文字)

  • 列表1
  • 列表2
  • 列表3

注意：此处都需在-+=*后面加空格，区别于上面的斜体、粗体。两个 *号之间直接写内容

有序列表（数字后面加点，）

  1. 列表内容1
  2. 列表内容2
  3. 列表内容3
  4. 列表内容4

注意：数字加点后面需要空一格再加内容，如上4就失效了

列表嵌套（下一级敲三个空格）

  • 一级无序列表1
  • 一级无序列表2
    1. 二级有序列表1
    2. 二级有序列表2
      • 三级无序列表1
      • 三级无序列表2
    3. 二级有序列表3
  • 一级无序列表3
  • 一级无序列表4

9、表格写法

表头 表头 表头
内容 内容 内容
内容 内容 内容

第一行是表头 用丨隔开
第二行呀用“-”将表头和内容隔开，其中默认的字体是居左的，在-的两边加:号会让文字居中，在-的右边加:会让文字居右。
